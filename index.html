Nepscript ka kei Commands Haru:

General

bhana and suchana are the syntax for strarting a code. bhana Writes the code in console, while suchana Writes the code in the a Alerting Box. While using this syntax, we have to use ;e at the end of the code.

bhana hello, from prashant ;e
Suchana hello prashant ;e

Variables

manau and lekha are the syntax used to assume a variable. manau helps to insert variable at writting code, while lekha inputs user data. we have to end the lekha tag with ;e ra is the syntax used to concatinate the variables.

manau age = 5
manau name = lekha what is your name? ;e
bhana age ;e // simply bhana and ;e in manau varaible
bhana< "hello" ra name ;> //bhana< and ;> because input variable

Conditional

yedi checks if the given statement is true xa bhane is written at the end of yedi yata is used for multiple statement. xa bhane should be written to yata also. natra is used if none of the statements are true. ;q is used at end of the code.

yedi name="Prashant" xa bhane
